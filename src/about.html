About

This is a web based yet full featured jabber client. Using the form above you can log into any public jabber (xmpp) service.
You can also use the local jabber server to create your own jabber account. Domains available are: 'jwchat.org', 'muckl.org', 'x-berg.de' and 'open-im.com'. But of course you could also use any other jabber server that allows public registrations.
